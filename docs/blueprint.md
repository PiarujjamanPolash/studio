# **App Name**: AgencySplit

## Core Features:

- User & Agency Authentication: Secure email and password-based login and registration for individual users, ensuring each user is associated with a single agency for streamlined management.
- Partner & Profit Share Management: Establish and update agency details, and manage partner profiles including their individual profit share percentages, with validation to ensure percentages sum to 100%.
- Comprehensive Transaction Tracking: Record, categorize, edit, and delete all income and expense transactions with fields such as amount, date, description, category, project, and the partner who handled it. Offers filtering options by month, type, handled-by partner, or project.
- Dynamic Profit & Balance Calculation: Automate the calculation of total income, expenses, profit, individual partner's allocated profit shares, and their net balances for any specified time range.
- Settlement Suggestion Algorithm: An optimized function that processes current partner balances to suggest the minimal number of transfers required to settle accounts, simplifying partner payouts.
- Settlement Recording: Log actual partner-to-partner settlement payments, including sender, receiver, amount, and date, which then automatically adjust outstanding balances for accurate reporting.
- Agency Dashboard: A consolidated dashboard view presenting key financial metrics (income, expenses, profit), real-time partner balances, suggested settlements, and a quick overview of recent transactions.

## Style Guidelines:

- Primary brand color: A deep, professional blue (`#3B3BDA`). This color embodies trust, stability, and intelligence, ideal for financial and administrative tools.
- Background color: A subtly desaturated, cool-toned off-white (`#F1F1F6`). This provides a clean, neutral canvas that enhances readability and keeps the interface feeling light and open.
- Accent color: A vibrant yet serene sky blue (`#40BAE7`). This analogous accent hue provides visual highlight for interactive elements and call-to-actions, maintaining visual harmony with the primary blue while offering excellent contrast.
- The 'Inter' sans-serif typeface is recommended for all text elements. Its objective, highly legible, and modern design makes it perfectly suited for conveying financial data and ensuring clarity in an administrative interface, whether for headlines or body text.
- Employ a consistent set of minimal, outline-style icons. Icons should be clear and instantly recognizable to support intuitive navigation and understanding within the professional admin layout.
- Implement a clean, card-based layout for summary information on dashboards and a responsive table format for detailed transaction lists. The design must be adaptable for seamless use across both desktop and mobile devices, prioritizing content legibility and ease of interaction.
- Incorporate subtle, quick animations for state changes, such as button clicks, form submissions, or data loading. These animations should provide user feedback without being distracting, enhancing the overall fluidity and responsiveness of the application.
